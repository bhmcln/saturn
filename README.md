# Purpose

The purpose of this repo is to provide a shadcn/ui style respository for time-based UI elements, such as: daily, weekly, and monthly rostering; scheduling for an individual, scheduling for an organization, etc. This will extend into distinctions between shifts, tasks, employees/team-members, etc. 

A great deal of care will be put in to make these as minimal, low-overhead as possible to make them as reusable as possible for a wide array of usecases.
